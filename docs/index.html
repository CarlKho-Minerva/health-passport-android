Tech Stack

  • Nexa AI SDK (v0.0.24) — On-device model inference
  • OmniNeural-4B — Vision Language Model for document scanning
  • Qwen3-4B / Llama-3.2-3B — LLM for health Q&A
  • PaddleOCR — OCR fallback for text extraction
  • Parakeet-TDT — On-device speech-to-text
  • EmbedGemma — Embeddings for semantic search
  • Kotlin + Android SDK 36

Requirements

  • Android 8.1+ (API 27)
  • Snapdragon 8 Gen 2 or newer recommended for NPU features
  • 4GB+ RAM recommended
  • ~2GB storage for AI models (downloaded on first use)
